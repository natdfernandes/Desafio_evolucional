# Olá, bem-vindo(a) ao desafio técnico.
## Sobre o projeto:
_O projeto foi desenvolvido com o intuito de ajudar uma instituição de ensino educacional focado em estudantes e professores._
_Conta com um design intuitivo e que facilite o dia-a-dia com buscas, que permite ao usuário editar e salvar novas informações de alunos, professores e turmas, com navegação para a tela 1 e tela 2._
_E possui gráficos informativos para obter informações importantes que podem ser usados para meios educacionais._

### Tecnologias utilizadas
- HTML
- CSS
- JAVASCRIPT
